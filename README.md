# PVP Certification Explorer (India)

R-based exploratory project on India's Plant Variety Protection (PVP) certificates.

Focus:
- Who dominates PVP certificates over time (public vs private).
- How crop-wise and firm-wise concentration evolves.

This repo is part of a broader portfolio on agritech, biotechnology regulation, and tech-policy analytics.

---

## Data

- **Source**: India's PVP registry --> published lists of granted certificates as per the Protection of Plant Varieties & Farmers' Rights Act (PPVFRA), 2001.

---

## Methods

All analysis is in **R**, using:

- `tidyverse` for data cleaning and manipulation.
- `janitor` for name cleaning.
- `lubridate` for dates.
- `ggplot2` for visualisation.
- `plotly` for interactive graphs.
- `targets` for a full reproducible pipeline.
